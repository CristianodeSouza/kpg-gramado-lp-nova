# Extração de Design System - KPG Imóveis

**Instrução:** Use este prompt na extensão Chrome do Claude enquanto você acessa https://www.kpgimoveis.com.br/

## 📋 Prompt para Extensão Chrome

Copie e cole este prompt na extensão Chrome do Claude:

```
Você está no site https://www.kpgimoveis.com.br/

Por favor, extraia e liste EXATAMENTE as seguintes informações de design:

## CORES PRINCIPAIS
Liste todas as cores que vê no site (peça para inspecionar com F12 → elementos):
- Cor de fundo principal: (descrever e tentar pegar hex)
- Cor de text principal: (hex)
- Cor de links: (hex)
- Cores de botões (todos os tipos): (hex)
- Cores de destaque/accent: (hex)
- Qualquer outra cor importante: (hex)

## TIPOGRAFIA
- Font-family usada (qual nome exato da fonte?)
- Tamanhos de título H1, H2, H3: (em px ou rem)
- Tamanho de texto corpo: (em px ou rem)
- Tamanho de texto pequeno/label: (em px ou rem)
- Font-weight padrão (normal, bold, 400, 700, etc)

## BOTÕES
Descreva cada tipo de botão que vê:
- Botão primário: cor fundo, cor texto, bordas, tamanho, padding
- Botão secundário: (mesmas informações)
- Botão terciário: (mesmas informações)
- Estados dos botões: hover, active, disabled (mudam de cor? como?)

## ESPAÇAMENTO
- Margin padrão entre seções: (pixels/rem)
- Padding interno dos containers: (pixels/rem)
- Espaço entre elementos: (pixels/rem)

## OUTROS COMPONENTES
- Cards: tem borda? sombra? padding? raio das bordas?
- Inputs/Formulários: cor borda, cor fundo, tamanho fonte
- Ícones: tamanho, cor
- Imagens: tamanho típico, proporção

## LAYOUT
- Largura máxima do container (max-width): (pixels)
- Breakpoints para mobile/tablet/desktop: (se houver)
- Grid ou Flexbox usado?

Forneça tudo em formato estruturado, com valores exatos que puder encontrar.
```

---

## ✅ Como Executar

1. Acesse: https://www.kpgimoveis.com.br/
2. Abra a extensão Chrome do Claude (canto superior direito)
3. Cole o prompt acima na caixa de texto
4. Clique em "Enviar"
5. Copie a resposta completa
6. Paste aqui em baixo na seção "RESULTADO"

## 📌 Informações Alternativas

Se não conseguir pela extensão, você pode:

1. **Inspecionar no navegador (F12)**
   - Clicar em elemento do site
   - Ir em "Styles" ou "Computed"
   - Copiar hex das cores, fonts, tamanhos

2. **Usar DevTools**
   - Selecionar elementos
   - Copiar CSS
   - Colar aqui

3. **Descrever manualmente**
   - Describe as cores aproximadas
   - Descrever os botões
   - Tipo de fonte (sans-serif? serif?)
